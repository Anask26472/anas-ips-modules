# Enterprise Gaps That Still Remain

This repo is closer to an enterprise-style product, but some categories still remain outside the scope of a single student Python module:

- true inline drop path in the Linux kernel/network stack
- protocol-aware HTTP/DNS/TLS parsers and reassembly
- HA clustering / fail-open and fail-close modes
- signed update pipeline for rules/models
- full RBAC with users, sessions, password auth, and secret rotation
- SOC console / multi-node management server
- benchmarked packet-loss and line-rate validation

These are now documented as explicit future work rather than being implied silently.
